Background

  • We are working with social workers and christian missionaries in Brazil
  • Our client manually study and record conditions and development of local communities
  • They are doing this via CAQDAS

Goal

  • Our goal is to help our client visualize and plan strategies for successful interventions

Project

  • We created a plugin in Obsidian to automatically record social factors and does georeferencing of it that is lacking in most CAQDAS tools

Old

Old Software

  • CAQDAS (Computer Assisted Qualitative Data Analysis Software)
  • This is software that provides tools for the user to organize and manage their data
  • The specific software client uses Socio Atlas
  • The problem with this is that it is difficult to use

New

  • Move from SocioAtlas to Obsidian
  • Use a software that will organize our data better than the previous software used by the client
  • Our plugin seeks to organize the social factors within their data with one click

Improvements

  1. Continue implementing multiple analysis: interactive visualization map
  2. Implement AI system to help our simple analysis: automatically read data and get factors automatically
  3. Improve our georeferencing
  4. Create the obsidian plugin in a website
